<h1>Реализация верстки и виджета для заказа билетов: фильтры, календарь.</h1>
<p>Очень-очень старое тестовое для уже не существующей IT-компании. Макет был в формате JPEG, все изображения и текст пришлось вытаскивать из него. Одним из условий был запрет на использование сторонних библиотек и плагинов, поэтому все - на чистом JS. Позднее я уже для себя сделала мобильную версию.</p>
<h2>Особенности проекта</h2>
<ul>
  <li>Языки: HTML5, CSS3, JavaScript</li>
  <li>Семантическая разметка</li>
  <li>Адаптивность: версии для ПК, планшетов и мобильных. Desktop first</li>
  <li>Интерактивность: форма, фильтры</li>
</ul>
